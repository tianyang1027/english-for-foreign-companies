# 外企程序员常用英语 - 日常沟通

> 本文档整理了程序员在外企工作中常用的日常沟通英语。

| 英文 | 中文意思 | 使用场景 / 例句 |
|------|----------|----------------|
| How’s it going? | 最近怎么样？ | 问候同事，轻松口语 |
| I’m swamped / overloaded | 我工作很忙 | 日常聊天，说明忙碌状态 |
| I’m on it | 我正在处理 | 回应任务请求 |
| I’m blocked / stuck | 我遇到阻碍了 | 告诉团队你被卡住，需要帮助 |
| Can you give me a hand? | 你能帮我一下吗？ | 请求帮助 |
| No worries | 没关系 / 没问题 | 回复感谢或道歉 |
| Let’s catch up later | 我们之后再聊 | 暂时结束讨论或安排下一次沟通 |
| Thanks for the heads-up | 谢谢提醒 | 感谢别人提前通知 |
