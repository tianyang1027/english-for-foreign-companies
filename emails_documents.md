# 外企程序员常用英语 - 邮件 / 文档

> 本文档整理了程序员在外企写邮件和文档中常用英语。

| 英文 | 中文意思 | 使用场景 / 例句 |
|------|----------|----------------|
| FYI | 供参考 | 邮件里提供信息 |
| ASAP | 尽快 | 请求快速完成 |
| Please advise | 请指导 / 给意见 | 邮件请对方提供意见 |
| Let me know your thoughts | 请告诉我你的看法 | 请求反馈 |
| Thanks in advance | 先谢谢了 | 邮件结尾礼貌用语 |
| Apologies for the delay | 抱歉回复晚了 | 邮件回复晚了 |
| CC / BCC | 抄送 / 密送 | 邮件功能常用 |
| Per our discussion | 根据我们讨论 | 邮件衔接上次会议或讨论 |
