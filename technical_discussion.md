# 外企程序员常用英语 - 技术 / 项目讨论

> 本文档整理了程序员在外企开发、项目讨论中常用英语。

| 英文 | 中文意思 | 使用场景 / 例句 |
|------|----------|----------------|
| Refactor | 重构 | 优化代码结构 |
| Bug / Issue | 问题 / 错误 | 描述代码或系统错误 |
| Optimize | 优化 | 提高性能或效率 |
| Deploy | 部署 | 上线或推送更新 |
| Rollback | 回退 | 撤销更新 |
| Merge / PR | 合并 / 代码提交请求 | Git、代码审查 |
| Sandbox / DEV / STG / PROD | 沙箱 / 开发 / 测试 / 生产环境 | 环境区分 |
| CI / CD | 持续集成 / 持续交付 | DevOps 术语 |
| ETA | 预计完成时间 | 任务或上线时间 |
| MVP | 最小可行产品 | 项目阶段术语 |
